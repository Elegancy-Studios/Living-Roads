# Living Roads

Living Roads transforms standard FiveM traffic into a dynamic and realistic driving environment.

Instead of predictable AI vehicles, roads become alive with different driver personalities, changing traffic conditions, realistic reactions, and natural road behaviour that evolves with time, weather, congestion, and surrounding events.

Drivers behave differently depending on personality archetypes — some drive cautiously, others become impatient, aggressive, distracted, reckless, or inexperienced. Vehicles react to traffic flow, overtake, hesitate, tailgate, cut in, brake unexpectedly, queue differently, and create more believable road interactions.

## Features

* Realistic AI driving behaviour
* Dynamic overtaking system
* Driver personalities and behaviour archetypes
* Tailgating, lane drifting & lane hogging
* Variable reaction times
* Honking & road frustration
* Sudden braking events
* Rubbernecking near incidents
* Traffic violations & poor driving habits
* Rolling stops & amber light running
* Queue cutting & road rage events
* Rush hour traffic simulation
* Night driving behaviour changes
* Weather-aware traffic responses
* Rain, fog & storm driving adaptation
* Emergency vehicle corridor system
* AI pulls over for lights & sirens
* Scene slowdown behaviour
* Adaptive traffic light system
* Emergency traffic light pre-emption
* Congestion waves & phantom traffic jams
* Dynamic convoy generation
* Street racing events
* Smoother acceleration & vehicle movement
* Configurable performance settings

## Installation

1. Drag `living_roads` into your resources folder
2. Configure `config.lua`
3. Add:

ensure living_roads

4. Start your server
5. Play

## Notes

Disable Debug in config unless troubleshooting.

Config.Debug = false

Built for servers wanting roads that feel less scripted and more alive.
